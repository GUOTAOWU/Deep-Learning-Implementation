本プロジェクトは、最新の物体検出モデルであるYOLO（You Only Look Once）シリーズの異なるバージョン（YOLOv8、YOLOv9、YOLOv11）を用いて、カスタムデータセットに対する学習および性能評価を行うものです。具体的には、まずデータセットに含まれる画像およびラベルファイルのファイル名を正規化する前処理を行い、データの整合性をチェックした上で、訓練データの一部を可視化して確認します。その後、各モデルを用いて学習を実行し、検証データセットおよびテストデータセットを用いてPrecision（適合率）、Recall（再現率）、mAP（平均適合率）といった主要な指標に基づいた定量的評価を行います。最後に、学習済みモデルを用いて実際の画像に対する推論を行い、バウンディングボックスの描画結果を可視化することで、YOLOv11をはじめとする各モデルの検出精度を比較・検証することを目的としています。
